# Car pricing

## Description

App for users to get an estimate for how much their car is worth based on the make/model/year/mileage

## Installation

```bash
$ git clone https://github.com/JulianSimpli/car-pricing-nestjs.git
$ cd car-pricing-nestjs
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
